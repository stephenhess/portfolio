Security Engineer

Posted June 28

Coinbase

San Francisco, Ca
United States

Description

We're looking for security software engineers to build secure customer product and internal employee systems. Security software engineers will be designing and building secure software from the ground up, as well as breaking and refining our current architecture.

Responsibilities

Strong desire to own projects from inception to completion

Build new features at Coinbase to make users and data more secure.

Proactively identify and fix security vulnerabilities in the platform

Provide expertise to other engineers in the designing and building of secure software

Requirements

Strong foundation and in-depth technical knowledge of security engineering, authentication and security protocols and applied cryptography

Passion for security, and a practical and balanced approach to security issues

Contributions to the security community is a huge plus

What to send

Links to your GitHub/Stack Overflow/Portfolio/something awesome you've built are good ideas.

A resume or LinkedIn profile

A brief answer to the following question: Where is Coinbase most vulnerable and how would you address it?

How to apply

Send an email to jobs@coinbase.com with the information detailed above.
